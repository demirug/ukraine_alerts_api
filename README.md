## Ukraine alerts API

Scrapping information from https://alerts.in.ua and represent data in friendly format 

---


## Installation
Client is based on `python 3.7`

To install dependencies use `pipenv` tool and run `pipenv install`

Setup  .env file
-

```
DEBUG=True

REDIS_HOST=localhost
REDIS_PORT=6379
```
